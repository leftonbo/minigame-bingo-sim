# 🎱 とあるミニゲーム用のシミュレーター

ブラウザで動作するシミュレーター。

## 機能

- **5x5 カード**: 1〜25の数字（13はフリーマス）
- **抽選**: 1回 / 10回 / 100回 / 1000回の一括抽選
- **ボーナス**: 13が出ると2マスがランダムにアクティブ化
- **ライン判定**: 縦5 / 横5 / 斜め2の計12ライン
- **統計表示**: ヒット率、当たり率、スコア分布など

## セットアップ

```bash
# 依存関係インストール
npm install

# 開発サーバー起動
npm run dev
```

http://localhost:5173 にアクセス

## ビルド

```bash
npm run build
```
